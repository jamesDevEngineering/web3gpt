# Quick Start

Follow these steps to generate and deploy a smart contract.

## Steps

### 1. Launch Web3GPT

Login with GitHub to persist chat history.

### 2. Prompt

Input a request to start building your smart contract, dapp, or utility.

```plaintext
Create a simple ERC20 token called 'Demo Token' with a total supply of 1000000
```

### 3. Deploy

Deploy to a supported chain by prompting the Agent.

```plaintext
Deploy the contract to Sepolia
```

You can also click `Deploy Contract` to deploy manually with a connected wallet.

### 4. Testing

Follow the explorer link to view and interact with the deployed contract. The contract source code should be automatically verified through Blockscout within 60 seconds of deployment.
